# Model Weights
Place your trained YOLOv8 model here as `best.pt`.

This file is excluded from git (see .gitignore) because it's a large binary (6MB+).

## How to get the model:
1. Train via Google Colab using `train_colab.py`
2. Download `best.pt` from the training output
3. Place it in this directory as `models/best.pt`
